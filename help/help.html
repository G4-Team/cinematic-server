Cinematic URLs Documentation

GET

http://127.0.0.1:8000/users/get/{id}/

Get a user

Status Code: 200 OK

Successful Response Example: 
[
    {
        "message": "SUCCESSFUL: user catch successfuly",
        "user": {
            "id": 1,
            "username": "janesmith",
            "email": "jane@example.com",
            "phone": null,
            "birthday": "2012-03-02",
            "created_at": "2024-06-06 19:28:11"
        }
    }
]
Status Code: 400 Bad Request

Failure Response Example: 
[
    {
        "message": "ERROR: {error message}"
        "user": "Not found"
    }
]
GET

http://127.0.0.1:8000/users/add?username={username}&email={email}&password={password}&birthday={yyyy-mm-dd}

Create a new user

Status Code: 201 Created

Successful Response Example: 
[
    {
        "message": "SUCCESSFUL: user created successfuly",
    }
]
Status Code: 400 Bad Request

Failure Response Example: 
[
    {
        "message": "ERROR: {error message}"
    }
]
